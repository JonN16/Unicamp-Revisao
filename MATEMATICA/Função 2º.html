Função 2º Grau

1. O que é uma Função 2º Grau?

Uma função de 2º grau é uma relação matemática onde a variável dependente (geralmente "y" ou "f(x)") varia de forma quadrática com a variável independente (geralmente "x"). Sua fórmula geral é:

f(x) = ax2 + bx + c

Em outras palavras, a função descreve uma parábola no gráfico cartesiano. Essa parábola pode ser côncava para cima ou para baixo, dependendo do valor do coeficiente a.

  • x: Variável independente.
  • f(x) ou y: Variável dependente.
  • a: Coeficiente quadrático, determina a concavidade da parábola.
  • b: Coeficiente linear, determina a inclinação da parábola.
  • c: Coeficiente constante, indica onde a parábola corta o eixo y.

3. Arcos de Função

a > 0 (Abertura voltada para cima)

Δ > 0: A função quadrática possui duas raízes reais distintas, ou seja, ela corta o eixo X em dois pontos. Este gráfico mostra a curva com os dois pontos de interseção no eixo X.

Δ = 0: A função quadrática possui uma única raiz real (raiz dupla), ou seja, o vértice da parábola está exatamente sobre o eixo X. O gráfico demonstra esta situação especial.

Δ < 0: A função quadrática não possui raízes reais, ou seja, a parábola não corta o eixo X. O gráfico ilustra a curva sem interseção com o eixo X.

a < 0 (Abertura voltada para baixo)

Δ > 0: A função quadrática possui duas raízes reais distintas. A parábola se abre para baixo e corta o eixo X em dois pontos. Este gráfico ilustra essas interseções.

Δ = 0: A função quadrática possui uma única raiz real (raiz dupla). O vértice da parábola está exatamente sobre o eixo X, com a abertura voltada para baixo. Este gráfico demonstra essa situação.

Δ < 0: A função quadrática não possui raízes reais, ou seja, a parábola não corta o eixo X. Aqui, a parábola com abertura para baixo não toca o eixo X em nenhum ponto.

4. Vértice da parábola

O vértice é o ponto onde a parábola atinge seu valor mínimo (a > 0) ou máximo (a < 0).

As fórmulas para encontrar o vértice são:

XV = -b / 2a YV = -Δ / 4a

5. Fórmulas

A equação quadrática geral é representada por:

ax2 + bx + c = y

Forma Fatorada

A forma fatorada de uma equação quadrática é:

y = a(x - x1)(x - x2)

Onde:

  • a: Coeficiente quadrático.
  • x1 e x2: As raízes da equação quadrática.

A fórmula para calcular as raízes de uma equação quadrática é dada por:

x = (-b ± √(b2 - 4ac)) / 2a

Essa fórmula é conhecida como Fórmula de Bhaskara, e o discriminante Δ = b2 - 4ac determina o número de soluções reais:

  • Δ > 0: Duas raízes reais distintas.
  • Δ = 0: Uma única raiz real (raiz dupla).
  • Δ < 0: Nenhuma raiz real.

Além disso, as propriedades da soma e do produto das raízes podem ser usadas para simplificar cálculos:

  • Soma das raízes: S = x1 + x2 = -b / a
  • Produto das raízes: P = x1 ⋅ x2 = c / a

Equações incompletas

Para equações quadráticas incompletas, onde um dos coeficientes (b ou c) é zero, é possível simplificar os cálculos:

1. Equação do tipo ax2 - bx = 0

Nesse caso, o primeiro valor de x será sempre 0, e o outro pode ser calculado isolando a variável:

Exemplo: 4x2 - 6x = 0

Fatorando: x(4x - 6) = 0

Soluções: x = 0 ou x = 6 / 4

2. Equação do tipo ax2 + c = 0

Nesse caso, podemos usar:

x = ±√(-c / a)

Exemplo: x2 + 9 = 0

Soluções: x = √(9 / 1) → x = ±3

7. Questões

Determine x pertencente aos reais tal que (x² – 100x)².(x² – 101x + 100)² = 0.

Ver Resposta

Explicação: A equação será igual a zero quando qualquer um dos fatores for igual a zero. Para resolver:

  • 1º Fator: x² - 100x = 0 → Podemos fatorar como x(x - 100) = 0. Assim, x = 0 ou x = 100.
  • 2º Fator: x² - 101x + 100 = 0 → Isso é uma equação quadrática. Fatorando, temos (x - 1)(x - 100) = 0, logo x = 1 ou x = 100.

Resposta: O conjunto solução é {0, 1, 100}, pois são os valores que tornam a equação verdadeira.

Calcule o valor de k de modo que a função f(x) = 4x² – 4x – k não tenha raízes, isto é, o gráfico da parábola não possui ponto em comum com o eixo x.

Ver Resposta

Explicação: Para que uma função quadrática não tenha raízes reais, o discriminante (Δ) deve ser menor que zero.

O discriminante é dado por Δ = b² - 4ac. Substituímos os coeficientes da função:

Δ = (-4)² - 4(4)(-k) = 16 + 16k

Para que não existam raízes, exigimos que Δ < 0:

16 + 16k < 0 → 16k < -16 → k < -1

Resposta: Para que a função não tenha raízes reais, k deve ser menor que -1.

Determine os valores de m, para que a função f(x) = (m – 2)x² – 2x + 6 admita raízes reais.

Ver Resposta

Explicação: Para que uma função quadrática tenha raízes reais, o discriminante (Δ) deve ser maior ou igual a zero.

Aqui, Δ = b² - 4ac. Os coeficientes são: a = (m - 2), b = -2, c = 6.

Substituindo, temos:

Δ = (-2)² - 4(m - 2)(6) = 4 - 24(m - 2)

Expandimos: Δ = 4 - 24m + 48

Simplificando: Δ = 52 - 24m

Para que a função tenha raízes reais, Δ ≥ 0:

52 - 24m ≥ 0 → 24m ≤ 52 → m ≤ 52/24 → m ≤ 13/6

Resposta: O valor de m deve ser menor ou igual a 13/6.

O gráfico da função quadrática definida por y = x² – mx + (m – 1), em que m ∈ R, tem um único ponto em comum com o eixo das abscissas. Determine y associado ao valor de x = 2.

Ver Resposta

Explicação: Se o gráfico tem um único ponto em comum com o eixo x, então Δ = 0. Vamos determinar m usando:

Δ = b² - 4ac = 0 → (-m)² - 4(1)(m - 1) = 0

Expandindo: m² - 4m + 4 = 0 → (m - 2)² = 0 → m = 2

Substituímos m = 2 na função:

y = x² - 2x + 1

Para x = 2: y = (2)² - 2(2) + 1 = 4 - 4 + 1 = 1

Resposta: Quando x = 2, o valor de y é 1.

(UNESP - 2017) Uma função quadrática f é dada por f(x) = x² + bx + c, com b e c reais. Se f(1) = –1 e f(2) – f(3) = 1, o menor valor que f(x) pode assumir, quando x varia no conjunto dos números reais, é igual a:

Ver Resposta

Explicação: Primeiro, determinamos b e c usando as condições dadas:

Para f(1) = -1: 1² + b(1) + c = -1 → b + c = -2

Para f(2) - f(3) = 1:

[2² + 2b + c] - [3² + 3b + c] = 1 → 4 + 2b + c - (9 + 3b + c) = 1

Simplificando: 4 + 2b + c - 9 - 3b - c = 1 → -5 - b = 1 → b = -6

Substituímos b = -6 em b + c = -2:

-6 + c = -2 → c = 4

A função é f(x) = x² - 6x + 4.

O menor valor de uma função quadrática ocorre no vértice: x_v = -b/2a = -(-6)/2(1) = 3.

Substituímos x = 3 em f(x): f(3) = (3)² - 6(3) + 4 = 9 - 18 + 4 = -5.

Resposta: O menor valor é -5.

(UERJ - 2016) Observe a função f, definida por: f(x) = x² - 2kx + 29 para x ∈ R. Se f(x) ≥ 4, para todo número real x, o valor mínimo da função f é 4. Assim, o valor positivo do parâmetro k é:

Ver Resposta

Explicação: Para que f(x) ≥ 4, o vértice da parábola deve estar em f(x_v) = 4. O vértice ocorre em x_v = -b/2a = -(-2k)/2(1) = k.

Substituímos x = k na função e igualamos a 4:

f(k) = k² - 2k(k) + 29 = 4

Simplificando: k² - 2k² + 29 = 4 → -k² + 29 = 4 → k² = 25

Como k é positivo, temos k = 5.

Resposta: O valor positivo de k é 5.
