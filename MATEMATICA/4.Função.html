Funções Matemáticas

1. Função Composta

A função composta é o resultado da combinação de duas ou mais funções, onde a saída de uma é utilizada como entrada para outra. Essa combinação é representada por (g ◦ f)(x) = g(f(x))

Exemplo

Dadas as funções f(x) = 2x - 5 e g(x) = 3x + 1, vamos calcular:

1. g(f(x)) Substitua f(x) em g(x):
g(f(x)) = g(2x - 5) = 3(2x - 5) + 1 g(f(x)) = 6x - 15 + 1 = 6x - 14

2. f(g(x)) Substitua g(x) em f(x):
f(g(x)) = f(3x + 1) = 2(3x + 1) - 5 f(g(x)) = 6x + 2 - 5 = 6x - 3

2. Função Inversa

A função inversa é aquela que desfaz o efeito de uma função original. Para uma função f(x) e sua inversa f-1(x), temos que f(f-1(x)) = x e f-1(f(x)) = x.

Passos para encontrar a inversa:

  1. Substitua f(x)por y.
  2. Troque x por y.
  3. Isolando y, você encontra a função inversa.

Exemplo

Dada a funçãof(x) = 3x - 1, vamos encontrar sua inversa:

y = 3x - 1 Troque x por y<:
x = 3y - 1 Isolando y:
x + 1 = 3y y = (x + 1) / 3 Logo, a função inversa é f-1(x) = (x + 1) / 3

3. Função Modular

A função modular retorna o valor absoluto de um número, ignorando seu sinal. Isso é representado por f(x) = |x|

Exemplo

Dada a função f(x) = |2x - 6|, analisamos dois casos:

  • Se x ≥ 3, f(x) = 2x - 6
  • Se x < 3, f(x) = -(2x - 6) = -2x + 6

Assim, a função é definida em partes:

f(x) = { 2x - 6, se x ≥ 3; -2x + 6, se x < 3 }

4. Função Exponencial

Uma função exponencial tem a forma f(x) = ax, onde a é uma base positiva e diferente de 1, e x é o expoente.

Exemplo

Dada a função f(x) = 2x

  • f(2) = 22 = 4
  • f(3) = 23 = 8
  • f(-1) = 2-1 = 1/2

f(f(1)) = 1

x2 - 4x + 1 = 0

2. -(x2 - 4x + 3) = 2
-x2 + 4x - 3 = 2
-x2 + 4x - 5 = 0
Resolva a equação quadrática:
x = [4 ± √(16 + 20)] / 2
x = [4 ± √36] / 2
x = [4 ± 6] / 2
x = 5 ou x = -1

Portanto, os valores de x são 2 + √3, 2 - √3, 5 e -1
