POTENCIALIZAÇÃO

1. Introdução

Potencialização é uma operação matemática que consiste em multiplicar um número por ele mesmo várias vezes.

É como se fosse uma multiplicação repetida.

2 + 2 + 2 + 2 + 2 + 2 + 2 + 2 = 2 * 2 * 2 * 2 = 16 = 28

2. Propriedades

  • Produto de Potências de Mesma Base: Para multiplicar potências de mesma base, mantém-se a base e soma-se os expoentes.
    • xm * xn = xm+n
  • Quociente de Potências de Mesma Base: Para dividir potências de mesma base, mantém-se a base e subtrai-se os expoentes.
    • xm / xn = xm-n
  • Potência de Potência: Para elevar uma potência a outra potência, mantém-se a base e multiplica-se os expoentes.
    • (xm)n = xm*n
  • Potência de um Produto: Para elevar um produto a uma potência, eleva-se cada fator da base à potência indicada.
    • (xy)n = xn * yn
  • Potência de um Quociente: Para elevar um quociente a uma potência, eleva-se o numerador e o denominador à potência indicada.
    • (x/y)n = xn / yn
  • Base Negativa com Expoente Par ou Ímpar: Se a base for negativa e o expoente for par, o resultado será positivo. Se a base for negativa e o expoente for ímpar, o resultado será negativo.

    • Se n for par: (-x)n = xn
    • Se n for ímpar: (-x)n = -xn

3. Propriedades Negativas

  • Potência de Expoente Negativo: Para elevar um número a um expoente negativo, inverte-se a base e troca-se o sinal do expoente.
    • x-n = 1 / xn
  • Fração de Expoente Negativo: Para elevar uma fração a um expoente negativo, inverte-se a fração e troca-se o sinal do expoente.
    • (x/y)-n = (y/x)n

4. Radiação

Radiciação é a operação inversa da potenciação. Ela extrai a raiz de um número elevado a um expoente.

  • Raiz Quadrada: A raiz quadrada de um número é o valor que, multiplicado por si mesmo, resulta no número original.
    • √x = y → y * y = x
  • Raiz Cúbica: A raiz cúbica de um número é o valor que, multiplicado por si mesmo três vezes, resulta no número original.
    • ³√x = y → y * y * y = x

  • Potência com Expoente Fracionário: Uma potência com expoente fracionário pode ser expressa como uma raiz. Por exemplo:
    • a2/3 = (³√a)2
  • Produto de Radicais: A multiplicação de raízes pode ser simplificada multiplicando os radicandos.
    • √x * √y = √(x * y)
  • Raiz por Potência: A raiz de uma potência pode ser simplificada elevando o radicando ao expoente dividido pelo índice da raiz.
    • (³√x)2 = (x1/3)2 = x2/3
  • Raiz de Frações: A raiz de uma fração pode ser expressa como a raiz do numerador dividida pela raiz do denominador.
    • ³√(2/3) = ³√2 / ³√3
  • Simplificação de Raízes: A raiz com uma potencia pode simplicar a potencia e a raiz
    • ⁶√(34) = ³√(32)
  • Radiciação de Radiciação: A raiz de uma raiz pode ser simplificada multiplicando os expoentes fracionários.
    • √(√x) = x1/4

MMC para Calcular Valor de Raízes

Para calcular o valor de raízes, é necessário encontrar o mínimo múltiplo comum (MMC) dos expoentes.

Por exemplo, para calcular a raiz quadrada de 800:

800 2
400 2
200 2
100 2
50 2
25 5
5 5
1

Portanto, a raiz quadrada de 800 é:

√800 = √(25 * 52) = 22 * 5 * √2 = 20√2

Calcule o valor de 25.

Exemplo 2

Exemplo 3

Ver Resposta

Ver Resposta

6. Simplifique a expressão (23 * 42) / (82).
