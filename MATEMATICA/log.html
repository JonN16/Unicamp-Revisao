Logaritmos

Definição de Logaritmo

O logaritmo é a operação inversa da exponenciação. A definição de logaritmo é dada por:

logaB = X, onde:

  • a = base do logaritmo
  • B = logaritmando (o número sobre o qual se realiza a operação)
  • X = logaritmo, o número que indica o expoente

Ou seja, a expressão logaB = X significa que aX = B. Exemplo:

log10100 = 2, pois 102 = 100

Propriedades dos Logaritmos

As principais propriedades dos logaritmos são:

  • loga1 = 0: O logaritmo de 1 na base a é sempre 0, porque qualquer número elevado a 0 é igual a 1.

    • log21 = 0, pois 20 = 1

  • logaa = 1: O logaritmo da base a na própria base é 1, porque qualquer número elevado a 1 é igual a si mesmo.

    • log1010 = 1, pois 101 = 10

  • loga(B.C) = logaB + logaC: O logaritmo do produto é a soma dos logaritmos.

    • log10(100 * 1000) = log10100 + log101000

    log10100000 = 2 + 3 = 5

  • loga(B/C) = logaB - logaC: O logaritmo do quociente é a diferença entre os logaritmos.

    • log10(1000 / 10) = log101000 - log1010

    log10100 = 3 - 1 = 2

  • logaBn = n.logaB: O logaritmo de uma potência é o expoente multiplicado pelo logaritmo da base.

    • log216 = log224 = 4 * log22 = 4

  • logaB = logcB / logca: O logaritmo pode ser alterado para uma nova base c usando esta relação. Isso é útil quando você precisa calcular logaritmos em bases diferentes.

    • log28 = log108 / log102

    log28 = 0.903 / 0.301 ≈ 3

Exemplo Prático de Logaritmo

Suponha que você tenha que resolver o logaritmo:

log5125

Sabemos que:

log5125 = X, e que 5X = 125.

Como 53 = 125, podemos concluir que X = 3.

Conclusão

O logaritmo é uma ferramenta poderosa em várias áreas da matemática e ciência, sendo muito útil para resolver problemas envolvendo crescimento exponencial, escalas de medição e muitos outros campos. Ao compreender suas propriedades, você poderá manipulá-los de maneira mais eficiente e entender melhor as relações matemáticas.

Questão 1 (Adaptada)

Resolva a equação:

log2(x) + log4(x) = 3

Ver Resposta

Resolução:

Primeiro, transformamos log4(x) para a base 2 usando a mudança de base:

log4(x) = log2(x) / log2(4) = log2(x) / 2

Substituindo na equação original:

log2(x) + log2(x) / 2 = 3

Multiplicando todos os termos por 2 para eliminar a fração:

2log2(x) + log2(x) = 6

Somando os termos:

3log2(x) = 6 ⟹ log2(x) = 2

Resolvendo para x:

x = 22 = 4

Resposta: x = 4

Questão 2 (Adaptada)

Ver Resposta

Resolução:

Transformamos log9(x) para a base 3:

log9(x) = log3(x) / log3(9) = log3(x) / 2

Substituindo na equação original:

log3(x) + log3(x) / 2 = 4

x = 38/3

Questão 3 (Adaptada)

Se log10(2) = a e log10(3) = b, expresse log10(72) em função de a e b.

Ver Resposta

Resolução:

Fatoramos 72:

72 = 8 × 9 = 23 × 32

Aplicamos a propriedade do logaritmo do produto:

log10(72) = log10(23 × 32) = log10(23) + log10(32)

Aplicamos a propriedade do logaritmo da potência:

log10(72) = 3log10(2) + 2log10(3)

Substituímos os valores dados:

log10(72) = 3a + 2b

Resposta: log10(72) = 3a + 2b

Questão 4 (Adaptada)

Resolva a equação:

log2(x2 − 4x + 4) = 3

Ver Resposta

Resolução:

Primeiro, resolvemos o logaritmo:

log2(x2 − 4x + 4) = 3 ⟹ x2 − 4x + 4 = 23

Simplificamos:

x2 − 4x + 4 = 8 ⟹ x2 − 4x − 4 = 0

Resolvemos a equação quadrática:

x = 4 ± √(16 + 16) / 2 = 4 ± √32 / 2 = 4 ± 4√2 / 2 = 2 ± 2√2

Verificamos as condições de existência do logaritmo:

x2 − 4x + 4 > 0 ⟹ (x − 2)2 > 0

Isso é verdade para todo x ≠ 2. Portanto, ambas as soluções são válidas.

Resposta: x = 2 + 2√2 ou x = 2 − 2√2

Questão 5 (Adaptada)

Se log10(5) = a, determine log10(0,02) em função de a.

Ver Resposta

Resolução:

Escrevemos 0,02 como uma fração:

0,02 = 2 / 100 = 1 / 50

Aplicamos a propriedade do logaritmo do quociente:

log10(0,02) = log10(1 / 50) = log10(1) − log10(50)

Sabemos que log10(1) = 0 e fatoramos 50:

log10(50) = log10(5 × 10) = log10(5) + log10(10) = a + 1

Substituímos:

log10(0,02) = 0 − (a + 1) = −a − 1

Resposta: log10(0,02) = −a − 1

Questão 6 (Adaptada)

Resolva a equação:

log2(x) + log2(x + 2) = 3

Ver Resposta

Resolução:

Usamos a propriedade do logaritmo do produto:

log2(x(x + 2)) = 3

Isso implica:

x(x + 2) = 23

x2 + 2x = 8

x2 + 2x - 8 = 0

Resolvemos a equação quadrática:

x = [-2 ± √(4 + 32)] / 2

x = [-2 ± 6] / 2

As soluções são:

x = 2 ou x = -4

Como logaritmo de número negativo não é definido, a solução é:

Resposta: x = 2

Questão 7 (Adaptada)

Resolva a equação:

log3(x - 1) + log3(x - 2) = 1

Ver Resposta

Resolução:

Usamos a propriedade do logaritmo do produto:

log3((x - 1)(x - 2)) = 1

Isso implica:

(x - 1)(x - 2) = 31

x2 - 3x + 2 = 3

x2 - 3x - 1 = 0

Resolvemos a equação quadrática:

x = [3 ± √(9 + 4)] / 2

x = [3 ± √13] / 2

As soluções são:

x = (3 + √13) / 2 ou x = (3 - √13) / 2

Ambas as soluções são válidas.

Resposta: x = (3 + √13) / 2 ou x = (3 - √13) / 2

Questão 9 (Adaptada)

Resolva a equação:

log5(x + 1) - log5(x - 1) = 1

Ver Resposta

Resolução:

Usamos a propriedade do logaritmo do quociente:

log5((x + 1) / (x - 1)) = 1

Isso implica:

(x + 1) / (x - 1) = 51

(x + 1) / (x - 1) = 5

Multiplicamos ambos os lados por (x - 1):

x + 1 = 5(x - 1)

x + 1 = 5x - 5

Isolamos x:

1 + 5 = 5x - x

6 = 4x

x = 6 / 4

x = 3 / 2

Resposta: x = 3 / 2
